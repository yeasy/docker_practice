# Docker 映像檔

在之前的介紹中，我們知道映像檔是 Docker 的三大組件之一。

Docker 在執行容器前需要本地端存在對應的映像檔，如果映像檔不存在本地端，Docker 會從映像檔倉庫下載（預設是 Docker Hub 公共註冊伺服器中的倉庫）。
